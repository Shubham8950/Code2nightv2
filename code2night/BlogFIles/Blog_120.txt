<p><b>vue-js-toggle-button -</b></p><p>It is a npm package available for implementing toggle buttons. For installing this package you have to follow these steps:-</p><p><b>Step 1:-</b></p><p>First of all you have to install npm package using&nbsp;</p><p><br></p><pre>npm i vue-js-toggle-button</pre><p>After you execute this command in the terminal, vue js toggle button package will be installed in your project. After it is completed we will move to second stop of importing the package and using in the component.</p><p><b>Step-2:-</b></p><p>So after the package is installed, you have to go to the <b>main.js</b> in your vue application and add these lines</p><p></p><pre>import ToggleButton from 'vue-js-toggle-button'
  
Vue.use(ToggleButton)</pre><p>Now , after adding these lines your vue application is ready to use <b>vue-js-toggle-button</b>. So now you have to go to the component where you want to add that toggle button.And add this line to your html.</p><p><br></p><pre>&lt;toggle-button @change="checkData($event )" :value="Boolean(tr.isActive)" :labels="{checked: 'Yes', unchecked: 'No'}" style="margin-left: 20px" /&gt;</pre><p>Now&nbsp; talking about this line, we are creating a toggle button and <b>checkData($event)&nbsp; </b>will be a method where we will capture the toggle button value on change.So you have to add this in your methods</p><pre>	methods: {	 
	     checkData( event) {
      let value = event.value; /*will return true or false*/
  
		    },
		}</pre><p>So , this is how you can get the data of your toggle button .For setting the data we have used this. You have to pass true or false to set the value.&nbsp;&nbsp;</p><pre style="line-height: 1.42857; letter-spacing: 0.238px; text-align: justify;">:value="Boolean(tr.isActive)"</pre><p>So this is how we can implement toggle button in our application.
